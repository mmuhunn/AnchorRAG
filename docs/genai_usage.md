# GenAI Usage

The paper uses:

- `gpt-4o-2024-08-06` for answer planning/generation and the primary semantic judge in the live rerun config.
- `claude-sonnet-4-6` as a secondary robustness judge.
- `text-embedding-3-small` for document and query embeddings.

Generative AI tools also assisted editing and consistency checking. Final content, experiments, analyses, and claims remain author-controlled.
