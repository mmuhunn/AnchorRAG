# Ablation 210Q Snapshot

This folder contains the final per-topic cached ablation summaries used to
rebuild Table 8.

- `source_summaries/`: one final topic summary CSV per topic.
- `source_judgments/`: one final topic-level judged JSONL per topic. These are
  used only to recover the `claims` column from `evaluable_sentences`.
- `multi_anchor_note.csv`: generated by `scripts/make_supporting.py` for the
  single-anchor vs. top-2-anchor comparison reported with Table 8.

Run:

```bash
python3.8 scripts/make_supporting.py --target table8
```
