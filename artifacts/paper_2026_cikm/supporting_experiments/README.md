# Supporting Experiments

These artifacts back robustness, ablation, external-validation, and diagnostic
claims that sit around the main paper tables.

- `mbfc_external_validation/`: external MBFC factuality rescoring for Table 5.
- `statistical_tests/`: paired bootstrap and Wilcoxon outputs.
- `source_prior_validation/`: MBFC source-prior validation for Figure 2.
- `ablation_210q/`: final seven-topic ablation snapshots for Table 8.
- `sensitivity/`: compact Section 5.3 values and appendix sensitivity artifacts.
- `cross_judge_agreement/`: GPT-4o/Claude agreement notes.
- `mssr_verifier_tradeoff/`: verifier and MSSR trade-off notes.
- `per_topic_diagnostics/`: qualitative case-study diagnostics.
- `top2_anchor_negative_result/`: single-anchor vs. top-2-anchor negative result.

Cached supporting outputs are regenerated by:

```bash
python3.8 scripts/make_supporting.py
```
